# Cybersecurity Capstone Project - Assignment 17

## Overview

This capstone project implements a comprehensive cybersecurity framework tailored for a simulated small business (SMB). The framework is designed to protect digital assets—including customer data, financial records, and intellectual property—from common threats such as ransomware, phishing, and data breaches.

## Framework Approach

This project follows the **NIST Cybersecurity Framework**, organized around five core functions:

1. **Identify** - Develop organizational understanding to manage cybersecurity risk
2. **Protect** - Implement safeguards to ensure delivery of critical infrastructure services
3. **Detect** - Develop and implement capabilities to detect cybersecurity events
4. **Respond** - Develop and implement appropriate activities to take action regarding a detected cybersecurity incident
5. **Recover** - Develop and implement appropriate activities to restore capabilities or services impaired due to a cybersecurity incident

## Project Structure

```
├── 01-Identify/              # Risk assessment and organizational context
├── 02-Protect/               # Protection controls and preventative measures
├── 03-Detect/                # Detection mechanisms and monitoring
├── 04-Respond/               # Incident response procedures
├── 05-Recover/               # Business continuity and disaster recovery
├── Documentation/            # Framework documentation and policies
├── Risk-Assessment/          # Risk analysis and threat modeling
└── Implementation-Guide/     # Step-by-step implementation instructions
```

## Key Objectives

- ✅ Comprehensive risk identification and assessment
- ✅ Development of security policies and procedures
- ✅ Implementation of technical and administrative controls
- ✅ Establishment of monitoring and detection systems
- ✅ Creation of incident response and recovery plans
- ✅ Documentation of the complete security posture
